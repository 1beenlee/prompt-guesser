# Prompt Guessr (가칭) — Product Spec (v1.0)

## 0. 목표/원칙
- 결과물은 **무조건 이미지(또는 gif/짧은 영상)** 기반
- 플레이는 **30초 내** 완료 가능한 데일리 퀴즈(Wordle-like 루프)
- 수익 발생 전까지 **운영비 0원**
- 배포/호스팅: **Cloudflare Pages**
- 앱 구조: **Next.js + 정적 우선(SSG)**
- 데이터/저장: 퍼즐 메타(JSON) + 이미지 파일을 **GitHub 레포에 버전관리**
- 통계/분석: **개인 통계는 LocalStorage**, 트래픽/이벤트는 무료 분석 도구(예: GA)
- 데일리 기준: **UTC 00:00**에 전세계 동일 퍼즐 공개
- v1.0 범위: **심플 모드만 출시** (Pro는 1.1+)
- 결과창: **정답/오답과 함께 ‘정답 프롬프트 전문(Full Prompt)’을 접힘(Expand) 형태로 즉시 공개**

---

## 1. 제품 개요
Prompt Guessr는 “결과 이미지”를 보고, 그 결과를 만들기 위해 LLM/이미지 모델에 준 **프롬프트를 추측**하는 데일리 게임입니다.
- 유저는 매일 1개의 이미지 퀴즈를 풀고
- 결과를 스포일러 없이 공유하며
- 연속 스트릭과 개인 통계를 누적합니다.

---

## 2. MVP(심플 모드) 게임 루프
### 2.1 사용자 플로우
1) 오늘의 이미지 제시  
2) 3개의 짧은 프롬프트 선택지(A/B/C) 중 하나 선택  
3) 즉시 결과(정답/오답) 표시 + Full Prompt(접힘) 제공  
4) 개인 통계/스트릭 표시  
5) 공유 텍스트를 클립보드 복사 / OS 공유  

### 2.2 규칙
- 선택지는 3개
- 프롬프트는 5~10초 내 읽을 수 있는 길이
- 오답 선택지에도 “틀린 요소”가 노골적으로 보이지 않도록 구성하되, 정답에만 포함된 **결정적 디테일**로 동일 결과 재현성이 갈리도록 설계
- 공유 메시지에는 “근접도” 문구 미포함(정답/오답만)

---

## 3. 핵심 기능 요구사항 (v1.0)
### 3.1 퍼즐(데일리) 제공
- UTC 00:00 기준으로 퍼즐이 자동 전환됨
- 퍼즐 데이터는 `puzzles/daily/YYYY-MM-DD.json`에서 로드

### 3.2 퀴즈 화면
- 이미지 1장 표시(웹 최적 포맷 권장: WebP/AVIF)
- 선택지 3개 표시(A/B/C)
- 선택 즉시 결과 표시(정답/오답)

### 3.3 결과 화면
- 정답/오답 배지
- “정답 프롬프트 전문(Full Prompt)” 섹션 (기본 접힘, Expand 시 표시)
- (선택) Negative Prompt / Model / Aspect Ratio 등 “생성 레시피” 표준 필드 노출
- 공유 버튼
- 개인 통계(승률, 연속 스트릭, 최근 N일 기록)

### 3.4 공유(스포일러 방지)
- 텍스트 공유 포맷(예시)
  - 정답:
    - `PromptGuessr #YYYY-MM-DD — ✅ (1/1)`
  - 오답:
    - `PromptGuessr #YYYY-MM-DD — ❌ (0/1)`
- 이미지 썸네일/정답 프롬프트 내용은 공유 텍스트에 포함하지 않음

### 3.5 개인 통계(LocalStorage)
- 저장 항목:
  - 날짜별 정답 여부
  - 스트릭(연속 성공일)
  - 총 플레이 수 / 총 정답 수
  - 마지막 플레이 일시

### 3.6 운영자 수동 큐레이션(중요)
- 특정 날짜(스폰서십/트렌드/공휴일/이슈 대응)에 맞춰
  - 특정 퍼즐 JSON + 이미지 에셋을 미리 커밋해 “예약 공개” 가능해야 함
- 운영자 개입은 “레포에 파일 커밋”으로 해결(운영비 0원 원칙 유지)

## 5. 비기능 요구사항
- 성능: 모바일에서 LCP 2.5초 내 목표(정적/캐시 적극 활용)
- 접근성: 선택지는 키보드/스크린리더 가능
- 안전: 금지 주제/상표/저작권 이슈 콘텐츠는 퍼즐 큐에서 배제
- 비용: 외부 DB/스토리지/유료 API 사용 금지(수익 전)

---

## 6. 로드맵(요약)
### v1.0 (Simple Daily)
- 데일리 1문제(3지선다)
- 결과 + Full Prompt 공개(접힘)
- 로컬 통계 + 공유

### v1.1 (Pro Mode)
- Pill 선택형(예: 15개 중 선택) + 점수(%)
- 점수 기반 피드백 문구(Perfect/Great/Good…)
- 공유: 점수/등급만(스포일러 방지), 게이지 형태

### v1.2+
- 유저 출제 모드 / 룸 모드(동시 시작) / 리더보드(단, 운영비 0원 원칙을 깨지 않는 방식부터)

---

## 7. 성공 지표(초기)
- D1 리텐션(소프트런치): 40%+
- 공유율(세션 대비): 20%+
- 평균 체류시간: 60~120초
- 오류율/신고율: 10k 세션당 < 1
